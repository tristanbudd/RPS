# RPS - Security Policy

Security is a top priority for the RPS project. If you discover a vulnerability, please report it via the methods below.

## Reporting a Vulnerability

Please use the GitHub Security Advisory reporting process:

1. Go to **[Security > Vulnerabilities > Report a vulnerability](https://github.com/tristanbudd/SIST/security)**
2. Click "Create a private vulnerability report"
3. Fill in the details of the vulnerability

**Alternative Contact:**
You can also reach out via email at `contact@tristanbudd.com` with "Security Vulnerability" in the subject line.

## Supported Versions

- `main` branch
- All released versions (tags)

## Timeline

I will respond to all security reports as quickly as possible. Please allow **7 business days** for an initial response and triage.

Please review our [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) for guidelines on respectful communication.
