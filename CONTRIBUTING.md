# RPS - Contributing

I appreciate you considering contributing to Rust Paste Server! Here's how you can help make this project better.

## How to Contribute

### Bug Reports

If you encounter a bug, please report it by creating an issue:

1. Go to **[Issues](https://github.com/tristanbudd/RPS/issues)**
2. Click "New issue"
3. Describe the bug and include steps to reproduce it

### Feature Requests

If you have an idea for a new feature:

1. Open an issue with the "Feature request" label
2. Describe the feature and why it would be useful

### Pull Requests

I welcome pull requests! Here's the general workflow:

1. Fork the repository
2. Create a feature branch (`git checkout -b feat/FeatureName`)
3. Commit your changes (`git commit -m 'Add FeatureName'`)
4. Push to the branch (`git push origin feat/FeatureName`)
5. Open a pull request

## Code Style

Please follow the existing code style in the project. Run `cargo fmt` before submitting a pull request to ensure proper formatting.

## Testing

All changes should pass the existing tests. You can run the tests with `cargo test`.
